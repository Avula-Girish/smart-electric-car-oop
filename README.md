# Smart Electric Car Simulation 🚗⚡

A Python project built using **Object-Oriented Programming (OOP)** to simulate a smart electric vehicle.

## Features ✨

- 🚗 Start the car
- 🔋 Battery level tracking
- 🛣️ Drive simulation (battery reduces while driving)
- ⚡ Charging simulation
- 🧬 Multiple Inheritance implementation
- 📊 Console dashboard display

## OOP Concepts Used 🧠

- Classes and Objects  
- Constructors (`__init__`)  
- Inheritance  
- Multiple Inheritance  
- Methods  
- State changes

## Example Simulation ▶️

```python
c1 = Car("BMW",150,80)

c1.start()
c1.drive(20)
c1.display()

c1.charge(20)
c1.display()
```

## Sample Output 🖥️

```text
BMW is started 🏎️

Battery: 80% → 60% after driving

Battery: 60% → 80% after charging
```

## Purpose 🎯

This project was built to practice modeling a real-world system using Python OOP concepts.
